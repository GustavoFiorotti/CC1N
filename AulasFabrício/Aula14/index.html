Aula de box

Na aula de hoje, temos os seguintes atributos:

  • Class: Mais usado, cria um grupo para variados caracteres, quando declarados com a classe. Ex.:
    No Html: <p class="texto_verde">Exemplo.</p>
    No CSS: .texto_verde{}
  • Id: Mesmo que a class, mas não é semânticamente aceita. No HTML: <p id="texto_verde">Exemplo.</p>
    No CSS: #texto_verde{}
  • Span: Para citar uma classe no meio do texto.
  • Div: Criar caixas com elementos dentro por toda a página.
  • Pseuelementos: Os pseudo-elementos são uma característica do CSS que permitem selecionar e estilizar partes específicas de um elemento HTML. Eles permitem referenciar comportamentos virtuais não tangíveis que não existem diretamente no HTML. ":" Cria uma pseudoclass em cima de um elemento do html, já o "::" cria um pseudoelemento em cima de uma pseudoclass.
